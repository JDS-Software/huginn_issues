# Commands Cheatsheet


| Command | Description | Default Keymap | Range |
| --- | --- | --- | --- |
| :HuginnInit | Initialize a .huginn configuration file | — | No |
| :HuginnConfig | Open the .huginn configuration file | — | No |
| :HuginnLog | Show the Huginn log buffer in a floating window | — | No |
| :HuginnCreate | Create a new Huginn issue at the current location | `<leader>hc` | Yes |
| :HuginnShow | Show issues for the current buffer | `<leader>hs` | No |
| :HuginnNext | Jump to the next issue location in the current buffer | `<leader>hn` | No |
| :HuginnPrevious | Jump to the previous issue location in the current buffer | `<leader>hp` | No |
| :HuginnAdd | Add a tree-sitter scope reference to an existing issue | `<leader>ha` | No |
| :HuginnRemove | Remove a tree-sitter scope reference from an existing issue | `<leader>hr` | No |
| :HuginnHome | Show all project files with issues | `<leader>hh` | No |
| :HuginnDoctor | Check issue integrity and interactively repair problems | — | No |
